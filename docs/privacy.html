Privacy Policy

Pinterest Fashion Trends Dashboard — Last updated: April 18, 2026

1. Introduction

Pinterest Fashion Trends Dashboard ("the Application") is a personal trend analysis tool that uses the Pinterest API to retrieve publicly available trending keyword data. This privacy policy explains what data we collect, how we use it, and your rights.

2. Data We Collect

The Application accesses the following data through the Pinterest API:

  • Trending keywords — publicly available trending search terms across fashion-related interest categories (e.g., women's fashion, beauty, home decor).
  • Growth metrics — week-over-week, month-over-month, and year-over-year percentage changes for trending keywords.
  • Regional data — trend data filtered by geographic region (e.g., US, UK, DE, FR).

We do not collect, store, or process any personal information, user accounts, browsing history, or personally identifiable information (PII) from Pinterest users.

3. How We Use the Data

The trend data retrieved from Pinterest is used solely for:

  • Displaying fashion trend analytics in a private dashboard.
  • Tracking keyword growth over time for market research purposes.
  • Visualizing trend patterns across regions and interest categories.

4. Data Sharing

We do not sell, share, lease, or distribute any data obtained through the Pinterest API to third parties. All data is used exclusively within the Application for personal analysis.

5. Data Storage

Trend data is stored in a secure database (Supabase) with encrypted connections. No personal user data is stored. Only aggregated, publicly available trend metrics are retained.

6. Third-Party Services

The Application uses the following third-party services:

  • Pinterest API — to retrieve trending keyword data. Subject to Pinterest's Privacy Policy.
  • Supabase — for database hosting. Subject to Supabase's Privacy Policy.

7. Cookies & Tracking

The Application does not use cookies, analytics trackers, or any form of user tracking technology.

8. Children's Privacy

The Application is not intended for use by children under the age of 13 and does not knowingly collect any information from minors.

9. Compliance

This Application complies with:

  • Pinterest Developer and API Terms of Service
  • Pinterest Developer Guidelines
